# Space Invaders OpenGL (Multiplayer)

A multiplayer implementation of classic Space Invaders using modern OpenGL.

For the original Single Player version, see:
[https://github.com/jrzanol/SpaceInvaders/tree/18316...](https://github.com/jrzanol/SpaceInvaders/tree/18316f2b55d54a7f2cc295b6c5adf993dfb20294)

## 🛠️ Tech Stack

* **OpenGL** - Rendering API
* **GLFW** - Windowing and context management
* **GLEW** - OpenGL extension loader
* **GLM** - Mathematics library
* **Assimp** - 3D model loading
* **ImGui** - GUI tooling
* **STB** - Image loading

## 🚀 Building

**Important:** Before compiling the main project, you must compile the Assimp library. Refer to instructions located in:
`assimp/Build.md`
